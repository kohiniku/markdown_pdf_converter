# Markdown to PDF Converter ユーザー向けマニュアル

## 1. このアプリでできること
- マークダウン形式の文章をPDF 化できます。
- タイトルページ（表紙）、フォントサイズ、用紙サイズなどを簡単に調整できます。
- 画像をドラッグ＆ドロップすると自動でアップロードされ、Markdown 内へ挿入されます。

## 2. アクセス方法
- ブラウザでアプリの URL（{URL}）を開きます。
- ログイン等は不要です。

## 3. 画面構成
1. **PDF出力設定パネル（左）**: フォントサイズや用紙サイズ、表紙の有無を指定します。
2. **ファイルアップロードエリア（左）**: `.md` ファイルをドロップまたは選択できます。
3. **Markdown 入力欄（左）**: 直接 Markdown を編集できます。画像をドロップすると自動で挿入されます。
4. **変換ボタン（右上）**: `Convert to PDF` を押すと変換が開始されます。
5. **エラー／成功メッセージ（右）**: 変換結果がここに表示されます。成功時はダウンロードボタンが出ます。
6. **プレビュー（右下）**: 入力した Markdown の仕上がりをリアルタイム確認できます。

## 4. 基本的な使い方
1. 左ペインに Markdown を入力、または `.md` ファイルをアップロードします。
2. 必要に応じてフォントサイズ・用紙サイズ・向きを変更します。
3. 表紙を付けたい場合は「Title page」を ON にし、タイトル・日付・名前を入力します。
4. プレビューでレイアウトを確認します。
5. `Convert to PDF` を押して変換します。
6. 成功メッセージ内の `Download` ボタンで PDF を保存します。

## 5. 画像の扱い
- 画像ファイル（PNG/JPG/GIF/WEBP/SVG）を マークダウン入力欄にドロップすると自動でアップロードされ、`![alt](url)` 形式が挿入されます。
- 画像サイズは PDF 生成時に WeasyPrint が読み込みます。大きすぎる場合は自動で縮小されることがあります。

## 6. 特殊な書式を使う
### チェックボックスを設置する
- `- [ ]` / `- [x]` と書くとチェックボックスが設置できます。
### 強制改ページを挿入する
- 任意のページ位置で改ページしたい場合は `[[PAGEBREAK]]` を入力すると強制改ページされます。
### 画像サイズを調整する
- 画像サイズを調整したい場合は `![代替テキスト](画像URL){width=50%}` や `...{width=200}` のように末尾に `{width=...}` を付けると割合やピクセルで制御できます。（※画像のheightはオリジナルと等比率で自動調整されます）
### 文字に色をつける
- 文字に色を付けたい場合は `<span style="color:#ff0000">赤字</span>`もしくは`<font color="red">...</font>` のようにHTML形式で記載してください。
### 分類別見出しを設置する
- `:::note` や `> [!NOTE]` などのを入力すると見出しが設置できます。
ほかにも`tip` `warning` `caution` `info` などが利用できます。
### Bootstrapの適用
- (上級者向け)Bootstrapのクラスも利用可能です。

## 7. エラーが出た時
| 状況 | 対処 |
|------|------|
| `Markdown content is required` | 入力欄が空です。Markdown を入力してください。 |
| `Only .md files are supported` | `.md` 以外のファイルをアップロードしていないか確認。 |
| `Image too large` | 画像ファイルが 10MB を超えています。サイズを縮小して再試行してください。 |
| `Conversion failed` / `Error generating PDF` | ネットワークや PDF エンジンの不具合の可能性があります。再度変換を試すか管理者に連絡してください。 |
